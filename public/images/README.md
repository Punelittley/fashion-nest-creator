# Папка для изображений товаров

## Как добавить свои изображения:

1. **Добавьте файлы изображений в эту папку** (`public/images/`)
   - Поддерживаемые форматы: JPG, PNG, WEBP
   - Рекомендуемый размер: 1200x1600 пикселей (соотношение 3:4)
   - Названия файлов должны быть на английском, без пробелов (например: `coat-black.jpg`)

2. **Обновите базу данных** через Backend (Cloud → Database)
   
   Пример SQL для одного изображения:
   ```sql
   UPDATE products 
   SET image_url = '/images/coat-black.jpg'
   WHERE name = 'Чёрное пальто';
   ```

   Пример SQL для нескольких изображений (слайдер):
   ```sql
   UPDATE products 
   SET images = ARRAY['/images/coat-black-1.jpg', '/images/coat-black-2.jpg', '/images/coat-black-3.jpg']
   WHERE name = 'Чёрное пальто';
   ```

## Структура папки:

```
public/images/
├── coats/           # Верхняя одежда
├── pants/           # Штаны и брюки
├── scarves/         # Шарфы
└── shoes/           # Обувь
```

## Примеры путей в базе данных:

- `/images/coat-black.jpg` - одно изображение
- `/images/coats/winter-coat.jpg` - изображение в подпапке
- ARRAY['/images/coat-1.jpg', '/images/coat-2.jpg'] - несколько изображений для слайдера

## Важно:

- Путь в базе данных должен начинаться с `/images/`
- После добавления изображений перезагрузите страницу каталога
- Изображения будут доступны сразу без перезапуска приложения

# Быстрый старт Backend сервера

## Установка зависимостей

```bash
cd server
npm install
```

## Запуск сервера

```bash
npm start
```

Или для разработки с автоперезагрузкой:

```bash
npm run dev
```

## После запуска

Сервер будет доступен на `http://localhost:3001`

База данных SQLite автоматически создастся в `server/fashion_store.db` и будет инициализирована с тестовыми данными:
- Админ: admin@test.com / admin123
- Категории товаров
- Несколько тестовых товаров

## Одновременный запуск

В корне проекта (для фронтенда):
```bash
npm run dev
```

В папке server (для бэкенда):
```bash
npm start
```
